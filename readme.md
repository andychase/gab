# GAB

Embeddable, <br />
Extendable, <br />
Minimal. <br />
Next generation forum software in PHP5.


## Installation

1. Run models/schema.sql in MySql
2. Configure sql username and password in index.php
3. Configure gab_config.php (optional)
4. Install source onto your server
5. Done
